# Quick Start Guide

Get started with ChestLockLite in just a few minutes.

## Basic Locking

### Lock Your First Chest

1. Place a chest in the world
2. Look at the chest
3. Run: `/cl lock`

That's it! The chest is now locked to you.

### Unlock a Chest

1. Look at your locked chest
2. Run: `/cl unlock`

## Using the GUI

The GUI is the easiest way to manage your chests:

1. **Shift + Right-click** any chest (while sneaking)
2. Click the "Lock Chest" button
3. Click "Set Password" if you want password protection

Alternatively, use `/cl gui` while looking at a chest.

## Adding Password Protection

1. Lock your chest: `/cl lock`
2. Set a password: `/cl password mypassword123`
3. Anyone with the password can now unlock it

## Trusting Players

Share access with friends without sharing passwords:

1. Lock your chest: `/cl lock`
2. Trust a player: `/cl trust PlayerName`
3. That player can now access your chest

## Common Commands

| Command | Description |
|---------|-------------|
| `/cl lock` | Lock the chest you're looking at |
| `/cl unlock` | Unlock your chest |
| `/cl password <pass>` | Set or enter a password |
| `/cl trust <player>` | Trust a player |
| `/cl gui` | Open the GUI menu |
| `/cl info` | View chest lock information |

## Tips

- Use the GUI (Shift + Right-click) for visual management
- Passwords can contain letters and numbers
- Trusted players don't need passwords
- Auto-lock on place can be enabled in config
